# Images Directory

Place your hero image here.

## Recommended Image Specifications

### Hero Image
- **Filename**: `shutterstock_2165450175.jpg.webp` or `hero-image.png`
- **Dimensions**: 1200x800px (3:2 aspect ratio)
- **File Size**: < 500KB (optimized for web)
- **Format**: JPG or PNG
- **Content**: Skin care, dermatology, or health-related imagery

### Alternative Images
You can also add:
- `hero-image-mobile.jpg` - Optimized for mobile (800x600px)
- `hero-image-tablet.jpg` - Optimized for tablet (1000x700px)

## Usage

The hero image will be displayed on the Welcome/Landing page with:
- Responsive sizing
- Lazy loading
- Fallback gradient background
- Smooth fade-in animation

## Adding Your Image

1. Place your image in this directory: `src/assets/images/`
2. Name it `shutterstock_2165450175.jpg.webp` or `hero-image.png`
3. The WelcomePage component will automatically use it
4. If the image is missing, a beautiful gradient background will be shown

## Image Optimization Tips

- Use tools like TinyPNG or ImageOptim to compress
- Convert to WebP format for better performance
- Ensure good contrast for text overlay
- Test on different screen sizes
